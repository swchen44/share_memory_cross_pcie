## 1. 功能為PCIE slave 要data給PCIE host, 利用DRAM memory 佔放buffer
## 2. 有一個DRAM memory buffer，可以被host 讀寫, 也可以被PCIE slave 讀寫, 
## 3. 當memory 被 PCIE slave 寫資資料後， 更新write pointer，PCIE slave利用PCIE MSI interrupt 通知host, 
## 4. 當host 讀DRAM memory buffer 後，判斷read pointer 和 write pointer是否有資料， 並更新read potiner，host 利用CCIF interrupt 通知slave
## 5. Slave 若還有資料，就繼續上面第3步驟

## 寫一個 emi_reader.c 模組 c code，可模擬在pcie host執行, buffer address 由caller 提供, 此module為pure module，提供read byte和block 可read多個block, block大小是define, 定義預設4096 bytes, 一定是2的冪次
## 寫一個 main.c，給一個global buffer, 大小為32 kbytes, emi_writter, emi_reader共用此global buffer，把address傳給 emi_writter, emi_reader 模組, 模擬以上功能
